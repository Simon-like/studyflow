# 本科毕业论文（设计）中期报告

## StudyFlow：基于番茄工作法的智能学习陪伴应用设计与实现

---

## 一、项目简介

### 1.1 项目背景

在移动互联网时代，大学生群体面临着学习专注力下降、时间管理能力不足、学习孤独感增强等问题。据统计，超过70%的大学生表示在学习过程中容易受到手机干扰，难以保持长时间专注。针对这一痛点，本课题设计并实现了 **StudyFlow**——一款面向大学生群体的智能学习陪伴应用。

### 1.2 项目定位

StudyFlow 的核心定位是**番茄钟任务管理 + AI学习助手 + 学习社区**三位一体的高效学习工具，旨在通过科学的时间管理方法、智能化的学习陪伴和社交化的学习氛围，帮助用户建立良好的学习习惯，提升学习效率。

### 1.3 技术架构

本项目采用 **前后端分离** 架构：

| 层级 | 技术栈 | 说明 |
|------|--------|------|
| **前端（Web）** | React 18 + TypeScript + Vite + Tailwind CSS | 响应式网页应用 |
| **前端（Mobile）** | React Native + Expo SDK | 跨平台移动应用 |
| **后端** | Spring Boot 3.2 + Java 21 + Spring Data JPA | 企业级后端服务 |
| **数据库** | PostgreSQL 16 | 主数据存储 |
| **缓存** | Redis 7.x | 会话管理、热点数据缓存 |
| **安全** | Spring Security + JWT | 认证授权 |
| **部署** | Docker + Docker Compose | 容器化部署 |

#### 2.1.2 用户认证模块（Auth Module）

实现了完整的用户认证流程，包括注册、登录、Token刷新、登出等功能。

**核心特性：**
- **手机号唯一登录**：用户仅可通过手机号+密码登录
- **自动生成账号体系**：注册时自动生成唯一用户名（格式：`user_时间戳_随机数`）和 8位数字 PIN 码
- **bcrypt 密码加密**：采用 bcrypt 算法（默认12轮）对用户密码进行哈希存储
- **双令牌机制**：AccessToken（2小时有效期）+ RefreshToken（7天有效期）
- **Token 黑名单**：使用 Redis 存储已登出的 RefreshToken，防止令牌重用攻击

**关键代码实现：**

```java
@Service
@RequiredArgsConstructor
public class AuthService {
    
    private final UserRepository userRepository;
    private final JwtTokenProvider tokenProvider;
    private final PasswordEncoder passwordEncoder;
    private final RedisTemplate<String, String> redisTemplate;
    
    @Transactional
    public TokenResponse register(RegisterRequest request) {
        // 1. 校验手机号唯一性
        if (userRepository.existsByPhone(request.getPhone())) {
            throw new ConflictException("该手机号已被注册");
        }
        
        // 2. bcrypt加密密码
        String passwordHash = passwordEncoder.encode(request.getPassword());
        
        // 3. 生成唯一用户名和PIN码
        String username = generateUniqueUsername();
        String pin = generateUniquePin();
        
        // 4. 创建用户实体
        User user = User.builder()
            .username(username)
            .phone(request.getPhone())
            .passwordHash(passwordHash)
            .pin(pin)
            .nickname(request.getNickname() != null ? 
                request.getNickname() : "用户" + pin.substring(4))
            .build();
        
        userRepository.save(user);
        
        // 5. 初始化用户连续学习记录
        userStreakRepository.save(new UserStreak(user.getId()));
        
        // 6. 生成JWT令牌对
        String accessToken = tokenProvider.generateAccessToken(user);
        String refreshToken = tokenProvider.generateRefreshToken(user);
        
        return new TokenResponse(accessToken, refreshToken, 
            tokenProvider.getAccessTokenExpiration(), userMapper.toProfile(user));
    }
    
    public TokenResponse login(LoginRequest request) {
        // 1. 通过手机号查找用户
        User user = userRepository.findByPhone(request.getPhone())
            .orElseThrow(() -> new UnauthorizedException("手机号或密码错误"));
        
        // 2. 验证密码
        if (!passwordEncoder.matches(request.getPassword(), user.getPasswordHash())) {
            throw new UnauthorizedException("手机号或密码错误");
        }
        
        // 3. 生成新令牌对
        return generateTokenResponse(user);
    }
    
    public TokenResponse refreshToken(String refreshToken) {
        // 1. 验证令牌签名和有效期
        if (!tokenProvider.validateToken(refreshToken)) {
            throw new UnauthorizedException("无效的刷新令牌");
        }
        
        // 2. 检查黑名单
        if (isTokenBlacklisted(refreshToken)) {
            throw new UnauthorizedException("刷新令牌已失效");
        }
        
        // 3. 解析用户信息并生成新令牌
        String userId = tokenProvider.getUserIdFromToken(refreshToken);
        User user = userRepository.findById(userId)
            .orElseThrow(() -> new UnauthorizedException("用户不存在"));
        
        return generateTokenResponse(user);
    }
    
    public void logout(String refreshToken) {
        // 将刷新令牌加入Redis黑名单，TTL为令牌剩余有效期
        long expiration = tokenProvider.getExpirationDateFromToken(refreshToken).getTime() 
            - System.currentTimeMillis();
        if (expiration > 0) {
            redisTemplate.opsForValue().set(
                "blacklist:token:" + refreshToken, "true", 
                Duration.ofMillis(expiration));
        }
    }
}
```

#### 2.1.3 番茄钟模块（Pomodoro Module）

番茄钟是 StudyFlow 的核心功能，实现了完整的番茄工作法计时逻辑。

**功能特性：**
- **双模式支持**：任务模式（绑定具体任务）和自由模式（纯计时）
- **状态管理**：running → paused/completed/stopped 的完整状态流转
- **自动清理机制**：检测并自动清理超过计划时长2倍的超时记录
- **结算触发统计**：番茄钟完成后自动更新任务状态、每日统计、连续学习天数

**核心业务逻辑：**

```java
@Service
@RequiredArgsConstructor
public class PomodoroService {
    
    private final PomodoroRepository pomodoroRepository;
    private final TaskRepository taskRepository;
    private final UserStreakRepository streakRepository;
    
    @Transactional
    public PomodoroRecord startPomodoro(String userId, StartPomodoroRequest request) {
        // 1. 自动清理超时记录
        cleanupStalePomodoros(userId);
        
        // 2. 检查是否有进行中的番茄钟
        if (pomodoroRepository.existsByUserIdAndStatus(userId, PomodoroStatus.RUNNING)) {
            throw new ConflictException("已有进行中的番茄钟，请先结束它");
        }
        
        // 3. 校验关联任务（如提供）
        if (request.getTaskId() != null) {
            taskRepository.findByIdAndUserId(request.getTaskId(), userId)
                .orElseThrow(() -> new NotFoundException("关联任务不存在"));
        }
        
        // 4. 创建番茄钟记录
        PomodoroRecord record = PomodoroRecord.builder()
            .userId(userId)
            .taskId(request.getTaskId())
            .plannedDuration(request.getDuration())
            .isLocked(request.getIsLocked() != null ? request.getIsLocked() : false)
            .status(PomodoroStatus.RUNNING)
            .startTime(Instant.now())
            .build();
        
        return pomodoroRepository.save(record);
    }
    
    @Transactional
    public PomodoroSettlement stopPomodoro(String userId, String pomodoroId, 
            StopPomodoroRequest request) {
        // 1. 查询进行中的番茄钟
        PomodoroRecord record = pomodoroRepository
            .findByIdAndUserIdAndStatus(pomodoroId, userId, PomodoroStatus.RUNNING)
            .orElseThrow(() -> new NotFoundException("未找到进行中的番茄钟"));
        
        // 2. 计算实际专注时长
        Instant now = Instant.now();
        int actualDuration = (int) Duration.between(record.getStartTime(), now).getSeconds();
        
        // 3. 更新番茄钟记录
        record.setStatus(request.getStatus());
        record.setEndTime(now);
        record.setActualDuration(actualDuration);
        record.setAbandonReason(request.getAbandonReason());
        pomodoroRepository.save(record);
        
        Task updatedTask = null;
        
        // 4. 如果是完成状态，触发级联更新
        if (request.getStatus() == PomodoroStatus.COMPLETED) {
            // 4.1 更新关联任务状态为进行中
            if (record.getTaskId() != null) {
                updatedTask = taskRepository.findById(record.getTaskId())
                    .map(task -> {
                        task.setStatus(TaskStatus.IN_PROGRESS);
                        return taskRepository.save(task);
                    })
                    .orElse(null);
            }
            
            // 4.2 更新每日番茄统计（Upsert操作）
            updatePomodoroDailyStats(userId, actualDuration, record.getTaskId() != null);
            
            // 4.3 更新用户连续学习天数
            updateUserStreak(userId);
        }
        
        // 5. 获取今日统计
        TodayStats todayStats = getTodayStats(userId);
        
        return new PomodoroSettlement(record, updatedTask, todayStats);
    }
    
    private void updateUserStreak(String userId) {
        LocalDate today = LocalDate.now();
        UserStreak streak = streakRepository.findByUserId(userId)
            .orElseGet(() -> new UserStreak(userId));
        
        // 如果今天已经学习过，不重复计算
        if (streak.getLastStudyDate() != null && 
            streak.getLastStudyDate().equals(today)) {
            return;
        }
        
        // 判断是否是连续学习（昨天有记录则连续）
        boolean isConsecutive = streak.getLastStudyDate() != null && 
            streak.getLastStudyDate().plusDays(1).equals(today);
        
        int newStreak = isConsecutive ? streak.getCurrentStreak() + 1 : 1;
        
        streak.setCurrentStreak(newStreak);
        streak.setLongestStreak(Math.max(streak.getLongestStreak(), newStreak));
        streak.setLastStudyDate(today);
        if (!isConsecutive) {
            streak.setStreakStartDate(today);
        }
        
        streakRepository.save(streak);
    }
    
    private void updatePomodoroDailyStats(String userId, int focusSeconds, boolean hasTask) {
        LocalDate today = LocalDate.now();
        
        pomodoroRepository.upsertDailyStats(userId, today, 
            stats -> {
                stats.setCompletedCount(stats.getCompletedCount() + 1);
                stats.setTotalFocusSeconds(stats.getTotalFocusSeconds() + focusSeconds);
                if (hasTask) {
                    stats.setRelatedTaskCount(stats.getRelatedTaskCount() + 1);
                }
            },
            () -> PomodoroDailyStat.builder()
                .userId(userId)
                .statDate(today)
                .completedCount(1)
                .totalFocusSeconds(focusSeconds)
                .relatedTaskCount(hasTask ? 1 : 0)
                .build()
        );
    }
}
```

#### 2.1.4 任务管理模块（Task Module）

实现了完整的任务生命周期管理，支持子任务、拖拽排序、软删除等特性。

**功能特性：**
- **CRUD 操作**：创建、查询、更新、删除（软删除）
- **状态流转**：todo → in_progress → completed/abandoned
- **子任务支持**：通过 parent_id 实现任务层级关系
- **拖拽排序**：display_order 字段支持自定义排序
- **今日任务筛选**：智能筛选今日待办、进行中、今日创建的任务

```java
@Service
@RequiredArgsConstructor
public class TaskService {
    
    private final TaskRepository taskRepository;
    private final TaskDailyStatRepository taskDailyStatRepository;
    
    @Transactional(readOnly = true)
    public PageResult<TaskDTO> getTasks(String userId, TaskQueryRequest query) {
        // 构建动态查询条件
        Specification<Task> spec = (root, criteriaQuery, cb) -> {
            List<Predicate> predicates = new ArrayList<>();
            predicates.add(cb.equal(root.get("userId"), userId));
            predicates.add(cb.isNull(root.get("deletedAt")));
            
            if (query.getStatus() != null) {
                predicates.add(cb.equal(root.get("status"), query.getStatus()));
            }
            if (query.getPriority() != null) {
                predicates.add(cb.equal(root.get("priority"), query.getPriority()));
            }
            if (StringUtils.hasText(query.getKeyword())) {
                predicates.add(cb.like(cb.lower(root.get("title")), 
                    "%" + query.getKeyword().toLowerCase() + "%"));
            }
            
            return cb.and(predicates.toArray(new Predicate[0]));
        };
        
        // 分页查询
        Pageable pageable = PageRequest.of(query.getPage() - 1, query.getSize(),
            Sort.by("displayOrder").ascending().and(Sort.by("createdAt").descending()));
        
        Page<Task> page = taskRepository.findAll(spec, pageable);
        
        return PageResult.of(page.map(taskMapper::toDTO));
    }
    
    @Transactional(readOnly = true)
    public List<TaskDTO> getTodayTasks(String userId) {
        LocalDateTime todayStart = LocalDateTime.now().with(LocalTime.MIN);
        LocalDateTime todayEnd = LocalDateTime.now().with(LocalTime.MAX);
        
        return taskRepository.findTodayTasks(userId, todayStart, todayEnd)
            .stream()
            .map(taskMapper::toDTO)
            .collect(Collectors.toList());
    }
    
    @Transactional
    public TaskDTO createTask(String userId, CreateTaskRequest request) {
        // 获取当前最大排序号
        Integer maxOrder = taskRepository.findMaxOrderByUserId(userId);
        
        Task task = Task.builder()
            .userId(userId)
            .title(request.getTitle())
            .description(request.getDescription())
            .category(request.getCategory())
            .priority(request.getPriority() != null ? request.getPriority() : TaskPriority.MEDIUM)
            .dueDate(request.getDueDate())
            .parentId(request.getParentId())
            .displayOrder((maxOrder != null ? maxOrder : 0) + 1)
            .isToday(true)
            .status(TaskStatus.TODO)
            .build();
        
        taskRepository.save(task);
        
        // 更新今日任务创建统计
        updateTaskDailyStats(userId, TaskAction.CREATE);
        
        return taskMapper.toDTO(task);
    }
    
    @Transactional
    public TaskDTO toggleTaskStatus(String userId, String taskId) {
        Task task = taskRepository.findByIdAndUserId(taskId, userId)
            .orElseThrow(() -> new NotFoundException("任务不存在"));
        
        TaskStatus newStatus = task.getStatus() == TaskStatus.COMPLETED 
            ? TaskStatus.TODO : TaskStatus.COMPLETED;
        
        task.setStatus(newStatus);
        task.setCompletedAt(newStatus == TaskStatus.COMPLETED ? Instant.now() : null);
        taskRepository.save(task);
        
        // 更新统计
        if (newStatus == TaskStatus.COMPLETED) {
            updateTaskDailyStats(userId, TaskAction.COMPLETE);
        } else if (task.getStatus() == TaskStatus.COMPLETED) {
            updateTaskDailyStats(userId, TaskAction.UNCOMPLETE);
        }
        
        return taskMapper.toDTO(task);
    }
    
    @Transactional
    public void deleteTask(String userId, String taskId) {
        Task task = taskRepository.findByIdAndUserId(taskId, userId)
            .orElseThrow(() -> new NotFoundException("任务不存在"));
        
        // 软删除
        task.setDeletedAt(Instant.now());
        taskRepository.save(task);
    }
    
    @Transactional
    public void reorderTasks(String userId, List<TaskOrderItem> orders) {
        // 批量更新排序（使用事务保证原子性）
        for (TaskOrderItem item : orders) {
            taskRepository.updateDisplayOrder(item.getId(), userId, item.getOrder());
        }
    }
}
```

```java
    @Transactional
    public void changePassword(String userId, ChangePasswordRequest request) {
        if (!request.getNewPassword().equals(request.getConfirmPassword())) {
            throw new BadRequestException("两次输入的新密码不一致");
        }
        
        User user = userRepository.findById(userId)
            .orElseThrow(() -> new NotFoundException("用户不存在"));
        
        // 验证当前密码
        if (!passwordEncoder.matches(request.getCurrentPassword(), user.getPasswordHash())) {
            throw new BadRequestException("当前密码错误");
        }
        
        // 更新密码
        user.setPasswordHash(passwordEncoder.encode(request.getNewPassword()));
        userRepository.save(user);
    }
    
    @Transactional(readOnly = true)
    public List<StudyCalendarDTO> getStudyCalendar(String userId, 
            LocalDate startDate, LocalDate endDate) {
        // 获取日期范围内的番茄统计
        List<PomodoroDailyStat> pomodoroStats = pomodoroDailyStatRepository
            .findByUserIdAndStatDateBetween(userId, startDate, endDate);
        
        // 获取任务统计
        List<TaskDailyStat> taskStats = taskDailyStatRepository
            .findByUserIdAndStatDateBetween(userId, startDate, endDate);
        
        // 构建日期映射
        Map<LocalDate, PomodoroDailyStat> pomodoroMap = pomodoroStats.stream()
            .collect(Collectors.toMap(PomodoroDailyStat::getStatDate, s -> s));
        Map<LocalDate, TaskDailyStat> taskMap = taskStats.stream()
            .collect(Collectors.toMap(TaskDailyStat::getStatDate, s -> s));
        
        // 生成完整日期范围
        List<StudyCalendarDTO> result = new ArrayList<>();
        for (LocalDate date = startDate; !date.isAfter(endDate); date = date.plusDays(1)) {
            PomodoroDailyStat pStat = pomodoroMap.get(date);
            TaskDailyStat tStat = taskMap.get(date);
            
            result.add(StudyCalendarDTO.builder()
                .date(date.toString())
                .focusMinutes(pStat != null ? pStat.getTotalFocusSeconds() / 60 : 0)
                .pomodoros(pStat != null ? pStat.getCompletedCount() : 0)
                .tasks(tStat != null ? tStat.getCompletedCount() : 0)
                .hasStudy(pStat != null && pStat.getCompletedCount() > 0)
                .build());
        }
        
        return result;
    }
    
    private UserStats calculateUserStats(String userId) {
        // 并行查询各项统计
        CompletableFuture<Integer> totalFocusMinutes = CompletableFuture.supplyAsync(() ->
            pomodoroRepository.sumActualDurationByUserIdAndStatus(userId, PomodoroStatus.COMPLETED)
                .map(seconds -> seconds / 60)
                .orElse(0));
        
        CompletableFuture<Long> totalPomodoros = CompletableFuture.supplyAsync(() ->
            pomodoroRepository.countByUserIdAndStatus(userId, PomodoroStatus.COMPLETED));
        
        CompletableFuture<Long> totalTasks = CompletableFuture.supplyAsync(() ->
            taskRepository.countByUserIdAndDeletedAtIsNull(userId));
        
        CompletableFuture<Long> completedTasks = CompletableFuture.supplyAsync(() ->
            taskRepository.countByUserIdAndStatusAndDeletedAtIsNull(userId, TaskStatus.COMPLETED));
        
        CompletableFuture<UserStreak> streak = CompletableFuture.supplyAsync(() ->
            userStreakRepository.findByUserId(userId).orElse(null));
        
        // 等待所有查询完成
        CompletableFuture.allOf(totalFocusMinutes, totalPomodoros, totalTasks, 
            completedTasks, streak).join();
        
        try {
            return UserStats.builder()
                .totalFocusMinutes(totalFocusMinutes.get())
                .totalPomodoros(totalPomodoros.get().intValue())
                .totalTasks(totalTasks.get().intValue())
                .completedTasks(completedTasks.get().intValue())
                .currentStreak(streak.get() != null ? streak.get().getCurrentStreak() : 0)
                .longestStreak(streak.get() != null ? streak.get().getLongestStreak() : 0)
                .build();
        } catch (Exception e) {
            throw new RuntimeException("统计计算失败", e);
        }
    }
}
```

#### 2.1.6 数据统计模块（Stats Module）

实现了多维度的数据统计功能，包括今日统计、周统计、学科分布等。

```java
@Service
@RequiredArgsConstructor
public class StatsService {
    
    private final PomodoroRepository pomodoroRepository;
    private final TaskRepository taskRepository;
    private final UserStreakRepository streakRepository;
    
    @Transactional(readOnly = true)
    public OverviewStatsDTO getOverviewStats(String userId, String period) {
        // 计算当前周期和上一周期的时间范围
        PeriodRange currentRange = PeriodRange.of(period);
        PeriodRange prevRange = currentRange.previous();
        
        // 查询当前周期统计
        PeriodStats current = getStatsInRange(userId, currentRange);
        PeriodStats previous = getStatsInRange(userId, prevRange);
        
        // 查询连续学习天数
        UserStreak streak = streakRepository.findByUserId(userId).orElse(null);
        
        return OverviewStatsDTO.builder()
            .focusMinutes(current.getFocusMinutes())
            .completedPomodoros(current.getPomodoros())
            .completedTasks(current.getTasks())
            .streakDays(streak != null ? streak.getCurrentStreak() : 0)
            .compareLastPeriod(CompareStatsDTO.builder()
                .focusMinutes(calculateChange(current.getFocusMinutes(), previous.getFocusMinutes()))
                .pomodoros(calculateChange(current.getPomodoros(), previous.getPomodoros()))
                .tasks(calculateChange(current.getTasks(), previous.getTasks()))
                .build())
            .build();
    }
    
    @Transactional(readOnly = true)
    public List<SubjectStatDTO> getSubjectStats(String userId, String period) {
        PeriodRange range = PeriodRange.of(period);
        
        // 查询该周期内已完成的番茄记录（包含任务分类）
        List<PomodoroRecord> records = pomodoroRepository
            .findCompletedWithTaskCategory(userId, range.getStart(), range.getEnd());
        
        // 按学科分类聚合
        Map<String, Integer> categoryMap = new HashMap<>();
        int totalSeconds = 0;
        
        for (PomodoroRecord record : records) {
            if (record.getTask() != null && record.getTask().getCategory() != null) {
                String category = record.getTask().getCategory();
                int seconds = record.getActualDuration() != null ? record.getActualDuration() : 0;
                categoryMap.merge(category, seconds, Integer::sum);
                totalSeconds += seconds;
            }
        }
        
        // 转换为DTO并计算百分比
        final int finalTotalSeconds = totalSeconds;
        return categoryMap.entrySet().stream()
            .map(entry -> SubjectStatDTO.builder()
                .category(entry.getKey())
                .focusMinutes(entry.getValue() / 60)
                .percentage(finalTotalSeconds > 0 ? 
                    Math.round((entry.getValue() * 100.0 / finalTotalSeconds) * 10) / 10.0 : 0)
                .build())
            .sorted(Comparator.comparing(SubjectStatDTO::getFocusMinutes).reversed())
            .collect(Collectors.toList());
    }
    
    private String calculateChange(int current, int previous) {
        if (previous == 0) {
            return current > 0 ? "+100%" : "0%";
        }
        double change = ((double) (current - previous) / previous) * 100;
        String sign = change >= 0 ? "+" : "";
        return String.format("%s%.1f%%", sign, change);
    }
}
```
